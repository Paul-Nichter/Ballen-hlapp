# Domain Configuration für Strohballen Verwaltung

Aktuelle Domain: test.com
Protokoll: https
Basis-URL: https://test.com

## Konfiguration ändern

Um die Domain zu ändern, bearbeite die Datei: lib/config.ts

Ändere die folgenden Konstanten:
- DOMAIN: Die Hauptdomain (z.B. "test.com", "example.com")
- PROTOCOL: Das Protokoll ("https" oder "http")

Die Änderungen werden automatisch in der gesamten Anwendung übernommen.

## Verwendung

Die Domain-Konfiguration wird verwendet für:
- SEO Metadata (Open Graph, Twitter Cards)
- Absolute URLs in E-Mails
- API Callbacks
- Canonical URLs
- Social Sharing Links

## Beispiele

Absolute URL generieren:
```typescript
import { getAbsoluteUrl } from '@/lib/config'

const dashboardUrl = getAbsoluteUrl('/dashboard')
// Ergebnis: https://test.com/dashboard

const apiUrl = getAbsoluteUrl('/api/orders')
// Ergebnis: https://test.com/api/orders
